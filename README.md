# Computatis Development Project

Toto je projekt, pro snažší vývoj webové aplikace [Computatis](https://github.com/kubajj/Computatis).

## Instalace

``` bash
# git clone této složky
git clone https://github.com/kubajj/Computatis.git

#přesun do složky
cd Computatis

# instalace projektu
npm install

# spuštění lokálně
npm run dev
```
